# MedTrack Project

Sistema de gerenciamento e acompanhamento de medicamentos desenvolvido nas disciplinas de Projeto Interdisciplinar de 
Engenharia da Computação I, III e IV da UFRPE — Unidade Acadêmica de Belo Jardim (UABJ).

## Estrutura do Projeto

### [MedTrack Mobile](https://github.com/MedTrack-Project/MedTrack-Mobile)
Aplicação Android nativa desenvolvida em Kotlin para gerenciamento de medicamentos, notificações, histórico e integração
com reconhecimento visual.

### [MedTrack Backend & Web]()
Sistema backend desenvolvido com Spring Boot, responsável pela API REST, autenticação, regras de negócio e persistência 
de dados, juntamente com a interface web em React.

### [MedTrack AI](https://github.com/MedTrack-Project/MedTrack-IA)
Serviço de inteligência artificial responsável pelo reconhecimento visual de medicamentos utilizando visão computacional
com Python e FastAPI.

## Tecnologias Utilizadas

- Kotlin
- Android
- Spring Boot
- React
- JavaScript
- Python
- FastAPI
- PostgreSQL
- Docker

## Organização
